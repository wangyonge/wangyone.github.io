hexo基本命令

Posted on 2018-01-24

日常更新

  • 本地新建文件名:hexo new “博客名”
  • 生成文档: hexo g -d
  • 运行本地服务器:hexo s -g ,浏览器中http://localhost:4000
  • 同步到github:hexo d -g

卸载hexo

  • 3.0.0版本执行:npm uninstall hexo-cli-g,
  • 旧版本执行:npm uninstall hexo -g

安装旧版本hexo

  • 先卸载当前版本,以2.8.3为例,执行:npm uninstall hexo@2.8.3 -g,再初始化并安装依赖和插件

安装hexo

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server

域名绑定

如果你不想使用yourusername.github.io，也可以绑定自己的域名，将域名的CNAME指向yourusername.github.io，然后在github下面新建一个CNAME文件，里面写上你的域名，然后就能通过域名直接访问到yourusername.github.io了。
